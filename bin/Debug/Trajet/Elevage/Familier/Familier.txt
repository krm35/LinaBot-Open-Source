Pour utiliser le bot Familier
1) mettre les familiers sur soit, sans aucun autre objet (sauf les objets déjà équipé).
2) se mettre dans la banque
3) dans "familier" mettre l'ID du familier puis la stats à augmenter.
4) connecter le bot, puis lancer le trajet, il suffit ensuite de laisser le bot ouvert.

- Le bot gére les capacitées accrue
- Le bot gére la corpulence des familiers.
- Le bot gére le doule repas.
- Le bot se reconnecte au plus tôt pour nourrir les familiers.
